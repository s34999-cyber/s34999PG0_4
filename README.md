Classes-
CafeCustomer: Contains a customer id, first name, last name, and email.
MenuItem: An item on the menu. Consists of a code, name, price, category, and a count of how many MenuItems are created in the program.
Order: assigns an incremental number to each newly made order. consists of a given customer, an arraylist of items ordered, an arraylist of just the names of items ordered, a paid status.
A simple program that allows for the creation of items for a cafe, customers, and the ability to process orders.
